## Problem4

### 요구사항 분석

- 연못에 말하는 엄마 개구리와 엄마가 하는 말은 무엇이든 반대로 말하는 청개구리가 살고 있다.
- 청개구리 사전을 바탕으로 주어진 `word`를 반대로 변환하여 반환하도록 한다.

### 구현 기능

1. 문자열을 순회하며 사전에 알맞은 문자로 변환한다.
    - 알파벳 외의 문자는 변환하지 않는다.
    - 알파벳 대문자는 알파벳 대문자로 변환한다.
    - 알파벳 소문자는 알파벳 소문자로 변환한다.
2. 변환된 결과를 반환한다.

### 구현 역할

> Converter

- 사전 정보를 알고 있고 문자열을 변환할 책임을 진다.
