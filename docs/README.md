### 초기설정
- [x] 입력받은 친구 관계 정보 friends 로 key 와 values 형태로 Map 생성

- [x] 입력받은 user key 를 통해 내 친구 찾기

- [x] 찾은 친구의 이름과 점수 저장
  - 친구이름이 이미 저장되어 있다면 + 10 
- [x] vistors 를 통해 Map에 이름과 점수 저장
    - 친구이름이 이미 저장되어 있다면 + 1
- [x] 점수가 가장 높은 순으로 정렬하여 최대 5명을 return
  - 점수가 같다면 사전순으로 정렬해서 return
