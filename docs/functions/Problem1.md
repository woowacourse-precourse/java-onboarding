## 구현할 기능
2개의 클래스로 구성
- 페이지 번호를 객체에 저장 및 조회
- 페이지 번호의 각 자리 숫자를 모두 더함 
- 페이지 번호의 각 자리 숫자를 모두 곱함
- 더한값과 곱한값중 큰 값을 찾음
- 왼쪽과 오른쪽 페이지 모두 1~3과정을 진행하고 둘중 큰 쪽의 값을 저장
- 2명의 사람중 최대값이 큰 사람을 찾음
- 입력받은 페이지 번호의 예외사항을 검사
### Player
- field 값 set, get 함수
- sumEachNum: 번호의 각 자리수 더하는 함수
- mulEachNum: 번호의 각 자리수 곱하는 함수
- calMaxNum: 왼쪽 오른쪽 페이지 각각 더한값과 곱한값중 큰 값을 찾고 왼쪽 오른쪽중 큰 값을 찾아 저장
### Game
- startGame- 입력받은 값을 player 객체에 저장하고 승자를 찾아 알맞은 값을 출력
- pageException- 페이지 번호 예외사항 체크
- winner- 두 사람중 승자나 무승부 여부 확인후 알맞은값 반환
