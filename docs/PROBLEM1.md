## 🚀 PROBLEM1 기능 구현 목록 
***

### 입력

- 책의 왼쪽, 오른쪽 페이지를 입력 받는다.
  - 왼쪽, 오른쪽 페이지가 연속된 숫자가 아니면 예외를 반환한다.

### 동작

- 입력받은 페이지의 각 자리 숫자를 더하거나, 모두 곱해 가장 큰 수를 구한다.
- 페이지 점수 계산에 따라 포비와 크롱의 큰 수를 비교한다. 
  - 포비가 이기면 1을 반환한다.
  - 크롱이 이기면 2를 반환한다.
  - 무승부면 0을 반환한다.
  - 그외 예외는 -1을 반환한다.

### 출력

- 비교 결과에 따라 게임의 결과를 출력한다.
