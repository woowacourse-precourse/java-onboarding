### 기능 목록
- [ ] 포비와 크롱의 페이지 리스트를 받아 게임을 시작한다.
- [X] 페이지를 검증한다.
  - [X] 페이지 개수 체크 (2여야 함)
  - [X] 페이지 범위 체크 (1 ~ 400)
  - [X] 페이지가 연속되는가? (차이가 1인가?)
  - [X] 페이지가 조건에 맞지 않는 경우 -1을 반환한다.
- [X] 페이지마다 max 값을 구해야 한다.
- [ ] 우승자를 구한다.
  - [ ] 포비가 이기면 1을 반환한다.
  - [ ] 크롱이 이기면 2를 반환한다.
  - [ ] 무승부일 경우 0을 반환한다.
