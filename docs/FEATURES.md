## Problem 7 기능 목록

1. **Map 자료형으로 친구 관계 그래프 생성**

2. **유저의 친구가 아닌 사람들의 점수판 생성**

3. **유저의 친구들과 친구인 사람들의 점수 계산(+10)**
4. **방문자들의 점수 계산(+1)**
