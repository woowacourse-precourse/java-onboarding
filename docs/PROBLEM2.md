## 🚀 기능 목록
1.입력된 암호문을 cut리스트에 알파벳 하나씩 add한다.

2.cut리스트에서 연속된 문자가 있을시 연속된 문자를 remove해준다.

3.더 이상 연속된 문자가 없으면 answer에 담아 리턴해 준다.

### 제한사항(예외 처리)

- cryptogram은 길이가 1 미만 1000 초과인 문자열 일때.
- cryptogram은 알파벳 소문자로만 이루어져 있지 않을때.
- 암호문에 공백이 있을때.

### 실행 결과 예시드

| cryptogram | result |
| --- | --- |
| "browoanoommnaon" | "brown" |
| "zyelleyz" | "" |
