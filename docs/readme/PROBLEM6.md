## 기능 분석
사용자 중 같은 글자가 연속적으로 포함된 경우 닉네임 사용제한 (**제이**엠 - 엠**제이** - **제이**슨)

두 글자 이상히면 모두 포함

결과 값은 중복되는 닉네임을 가진 사람의 이메일을 오름차순으로 정렬하고 중복은 제거
